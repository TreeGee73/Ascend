## Description

Choose a character and duel your way through various creatures and bosses to the top of the tower. This is a browser-based RPG game featuring turn-based combat, character progression, and a tower-climbing adventure.

---

## User Story

I want to kill some time killing random creatures in an attempt to reach the final boss (top of the tower). I want to be able to enter my player name, select a character or have a character selected for me, and then battle random opponents for virtual cash and prizes.

**Intended for:** Anyone who wants to play a simple web-based video game.

---

## Usage

This is an interactive browser-based RPG game featuring:
- Player name entry and character selection
- Turn-based combat system
- Random opponent encounters
- Virtual rewards (cash and prizes)
- Tower progression toward final boss
- Game state persistence

---

## Technology

* **JavaScript** (52.8%) - Game logic and API implementation
* **Handlebars** (19.5%) - Templating engine for dynamic views
* **CSS** (17.7%) - Styling and layout
* **HTML** (10.0%) - Structure and interface
* **Node.js** - Server-side runtime
* **Express.js** - Web application framework
* **Sequelize** - ORM for database management
* **MySQL** - Database
